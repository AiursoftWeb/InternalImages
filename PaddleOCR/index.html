📡 API 接口说明

POST /api/ocr — 执行文字识别

上传图片文件，返回识别出的文字内容及其位置坐标。


      

    

    

      POST /api/ocr/base64 — Base64 模式识别
      

        
以 JSON 格式提交 Base64 编码的图片，返回识别结果。

        

      

    

    

      GET /api/system — 获取系统状态
      

        
获取当前硬件设备、CUDA 版本及模型加载状态。
