📡 API 文档

所有接口均无需身份验证，可直接通过 HTTP 调用。Base URL 即本服务地址。

POST /api/tts — 文字转语音
发送表单数据，返回 WAV 音频文件。 
# 使用 curl
curl -X POST http://HOST:8000/api/tts \
  -F "voice=reporter-zh" \
  -F "text=你好，欢迎使用语音合成服务。" \
  --output speech.wav

# 使用 Python requests
import requests
resp = requests.post("http://HOST:8000/api/tts",
    data={"voice": "reporter-zh", "text": "你好，世界！"})
with open("speech.wav", "wb") as f:
    f.write(resp.content)
GET /api/voices — 获取声纹列表
返回 JSON 数组，每个元素包含 name、filename、size_kb。
curl http://HOST:8000/api/voices
# [{"name":"reporter-zh","filename":"reporter-zh.wav","size_kb":245.3}]
POST /api/voices — 上传声纹
上传 .wav 文件并指定声纹名称（表单字段 name + file）。
curl -X POST http://HOST:8000/api/voices \
  -F "name=my-voice" \
  -F "file=@/path/to/voice.wav"
DELETE /api/voices/{name} — 删除声纹
按名称删除指定声纹。
curl -X DELETE http://HOST:8000/api/voices/my-voice
GET /api/system — 系统信息
返回设备类型、GPU 型号、CUDA 版本等信息。
curl http://HOST:8000/api/system
GET /health — 健康检查
返回 {"status":"ok"} 或 {"status":"degraded"}。
curl http://HOST:8000/health
